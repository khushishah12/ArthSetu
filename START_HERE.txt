ARTHSETU AI — NEON EDITION
===========================

FIRST SETUP
1. Copy .env.example to .env.local.
2. Create a Neon project.
3. Enable Neon Auth in that project.
4. Add DATABASE_URL, NEON_AUTH_BASE_URL and a 32+ character
   NEON_AUTH_COOKIE_SECRET to .env.local.
5. Run database/NEON_SETUP.sql once in Neon SQL Editor.
6. Run: npm install
7. Start the ML service.
8. Run: npm run dev

LOCAL URLS
Website:   http://localhost:3000
Dashboard: http://localhost:3000/app/dashboard
ML docs:   http://127.0.0.1:8000/api/docs

DEMO MODE
The dashboard can still run with synthetic browser-only history before Neon
is configured.

DEPLOYMENT
- Next.js: Vercel
- PostgreSQL and Auth: Neon
- FastAPI ML service: Render
